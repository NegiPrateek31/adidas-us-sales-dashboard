# Adidas US Sales Dashboard

## Project Overview
This project showcases an interactive Power BI dashboard analyzing Adidas U.S. sales performance across products, regions, channels, and retailers. The objective is to transform raw sales data into meaningful, business-ready insights using KPI-driven visual analytics.

---

## Dataset
The dataset includes U.S. sales records with the following attributes:
- Sales revenue
- Operating profit
- Units sold
- Product categories
- Gender segmentation
- Sales channels (In-store, Outlet, Online)
- Retailers
- State-level geographic data
- Time-based sales information

---

## Tools & Technologies
- Power BI – Data modeling and dashboard creation  
- Microsoft Excel – Data source  

---

## Key Analysis & Features
- Tracked key performance indicators including total sales, operating profit, and units sold
- Analyzed monthly sales trends to identify seasonal patterns
- Compared performance across product categories and gender segments
- Evaluated sales contribution by channel (In-store, Outlet, Online)
- Performed retailer-wise analysis to identify top-performing partners
- Visualized state-level sales distribution using geographic mapping

---

## Business Insights
- Identified major revenue-driving channels and retailers
- Highlighted seasonal peaks and fluctuations in sales performance
- Compared demand trends across products and customer segments
- Enabled faster regional and partner-level performance comparison through interactive dashboards

---


## Outcome
This project demonstrates how **business intelligence tools** can be used to transform large sales datasets into **executive-level dashboards** that support data-driven decision-making.
